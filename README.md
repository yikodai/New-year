# 跨年祝福网站

一个功能丰富的跨年祝福分享网站，让用户可以写下祝福、查看他人的祝福以及观看跨年视频。

## 功能特性

### 🎉 主要功能
1. **写祝福** - 用户可以写下自己的新年祝福并提交
2. **看祝福** - 查看所有用户提交的祝福信息
3. **看视频** - 观看跨年相关视频

### 📱 技术特点
- 响应式设计，适配各种屏幕尺寸
- 现代化UI设计，节日氛围浓厚
- 前后端分离架构
- 数据持久化存储

## 安装和运行

### 1. 安装Node.js

首先需要在您的计算机上安装Node.js和npm。

- 访问 [Node.js官网](https://nodejs.org/zh-cn/) 下载并安装最新版本
- 安装完成后，在命令行中输入以下命令验证安装：
  ```bash
  node -v
  npm -v
  ```

### 2. 安装项目依赖

在项目根目录下运行：

```bash
npm install
```

### 3. 启动服务器

```bash
npm start
```

服务器将在 http://localhost:3000 启动。

### 4. 访问网站

- **首页**：http://localhost:3000
- **写祝福**：http://localhost:3000/write.html
- **看祝福**：http://localhost:3000/view.html
- **看视频**：http://localhost:3000/video.html

## 功能说明

### 1. 写祝福

- 输入您的名字（最多20个字符）
- 输入您的祝福内容（最多200个字符）
- 点击提交按钮发送祝福
- 祝福将被保存到服务器

### 2. 看祝福

- 查看所有用户提交的祝福
- 祝福按时间倒序排列（最新的在前）
- 每条祝福显示作者、内容和提交时间
- 可以点击刷新按钮更新祝福列表

### 3. 看视频

- 观看跨年相关视频
- 支持播放/暂停、静音/取消静音、全屏等功能
- 提供推荐视频列表

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **后端**：Node.js, Express.js
- **数据存储**：JSON文件

## 静态预览

如果您没有安装Node.js，也可以直接在浏览器中打开 `public/index.html` 文件查看网站的基本结构，但部分功能（如提交和查看祝福）需要后端支持。

## 自定义

### 添加视频

将视频文件放入 `public/videos/` 目录，并在 `video.html` 中更新视频源路径。

### 修改样式

编辑 `public/css/style.css` 文件可以自定义网站的外观和主题。

### 添加动画效果

在 `public/js/main.js` 中可以添加更多的JavaScript动画和交互效果。

## 注意事项

- 确保在运行服务器前安装了所有依赖
- 定期备份 `data/blessings.json` 文件以防止数据丢失
- 请勿在生产环境中直接使用此代码，建议添加更多的安全措施
